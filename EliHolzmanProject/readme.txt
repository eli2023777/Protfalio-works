
# Project Name ("protfalio works")

This project is a portfolio works project containing 6 projects for landing pages. Each project is built using HTML, CSS. Includes one project using SCSS
.and one project using Bootstrap

starting
To get started with this project, simply download the files and open the index.html file in your web browser.

Prerequisites
To run this project, you will need a web browser such as Google Chrome or Mozilla Firefox.

Installer
No installation required for this project. Just download the files and open the index.html file in your web browser.

Built with
Bootstrap - The web framework used
HTML - the markup language used
CSS - the style sheet language used
Sass - the preprocessor scripting language used
